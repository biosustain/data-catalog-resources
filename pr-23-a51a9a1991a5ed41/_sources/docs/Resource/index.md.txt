# Resource
A resource is a **data file** or **external link** that holds the actual data associated  with a dataset. 
Resources can be in any format, such as CSV, Excel, JSON, XML, PDF, image files or URLs.
Each dataset can have **multiple resources**, making your data accessible and usable for others.
